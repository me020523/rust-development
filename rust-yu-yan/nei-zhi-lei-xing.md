## Slice

* 代表collection变量的一段切片
* slice类型的变量对值没有ownership

不同数据类型的slice类型定义

| 类型 | slice |
| :--- | :--- |
| 整型 | &\[i8\], &\[u8\], &\[i16}... |
| 浮点型 | &\[f32\], &\[f64\] |
| 布尔型 | &\[bool\] |
| 字符型 | &\[char\] |
| String | &str |
| 自定义类型 | &\[类型\] |
